## 📘 Purpose

This repository documents my revision of core networking fundamentals required for transitioning into cybersecurity roles.

* TCP/IP vs OSI
* Ports and Protocols

To strengthen foundational networking knowledge essential for threat analysis, firewall configuration, and network security monitoring.

With prior experience in telecom and networking (PBX & SIP systems), this project reflects my structured revision of networking concepts before advancing deeper into cybersecurity.
