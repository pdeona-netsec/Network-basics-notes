# Subnetting Basics

## Overview

Subnetting divides a large network into smaller logical networks to improve performance and security.

---

## Key Concepts

* IP Address = Network ID + Host ID
* Subnet Mask defines network boundary
* Example:
  IP: 192.168.1.0/24
  Subnet Mask: 255.255.255.0

---

## Why Subnetting Matters in Security

* Limits broadcast domains
* Controls lateral movement of attackers
* Helps in network segmentation and zero trust architecture

---

## Example

If one subnet is compromised, segmentation prevents attackers from accessing the entire network.
