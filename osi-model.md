# OSI Model – Networking Basics Refresher

## Overview

The OSI (Open Systems Interconnection) model is a 7-layer conceptual framework used to understand how data travels across a network.

It helps in troubleshooting, security analysis, and protocol understanding.

---

## 7 Layers of OSI Model

1. Physical – Transmission of raw bits over cable
2. Data Link – MAC addressing and switching
3. Network – IP addressing and routing
4. Transport – End-to-end delivery (TCP/UDP)
5. Session – Session establishment and termination
6. Presentation – Data formatting and encryption
7. Application – User-facing services (HTTP, FTP, DNS)

---

## Importance in Cybersecurity

* Helps identify where attacks occur
* Example:

  * Phishing → Application Layer
  * ARP Spoofing → Data Link Layer
  * DDoS → Network/Transport Layer

Understanding OSI layers helps analysts detect and mitigate attacks efficiently.
