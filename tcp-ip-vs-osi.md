# TCP/IP vs OSI Model

## 📘 Overview

Both OSI and TCP/IP models describe network communication layers, but TCP/IP is the practical model used on the internet.

---

## Key Differences

| Feature      | OSI Model                  | TCP/IP Model          |
| ------------ | -------------------------- | --------------------- |
| Layers       | 7 Layers                   | 4 Layers              |
| Type         | Conceptual Model           | Practical Model       |
| Usage        | Learning & troubleshooting | Real-world networking |

---

## Layer Mapping

* OSI Application + Presentation + Session → TCP/IP Application
* OSI Transport → TCP/IP Transport
* OSI Network → TCP/IP Internet
* OSI Data Link + Physical → TCP/IP Network Access

---

## Cybersecurity Relevance

Understanding both models helps in:

* Packet analysis
* Firewall rule placement
* Identifying attack surface layers
