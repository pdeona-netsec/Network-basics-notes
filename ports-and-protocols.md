# Common Ports and Protocols

## Overview

Ports and protocols define how services communicate over a network. Understanding them is essential for firewall configuration and threat detection.

---

## Important Ports

| Port  | Protocol | Usage                  |
| ----- | -------- | ---------------------- |
| 20/21 | FTP      | File Transfer          |
| 22    | SSH      | Secure Remote Login    |
| 23    | Telnet   | Insecure Remote Access |
| 25    | SMTP     | Email Sending          |
| 53    | DNS      | Domain Name Resolution |
| 80    | HTTP     | Web Traffic            |
| 443   | HTTPS    | Secure Web Traffic     |
| 3389  | RDP      | Remote Desktop         |

---

## Cybersecurity Importance

* Open ports increase attack surface
* Example: Exposed RDP (3389) → Brute-force attacks
* Analysts monitor unusual port activity for intrusion detection
